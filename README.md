# Semantic HTML Practice

Learning semantic HTML for improvement of accessibility. Rebuilding real website using pure semantic HTML to learn structure.

## nav-practice

- Using aria-labels for screen readers.
- Referencing correctly.
- Links should be in a list.
- Heading hierarchy.

## contact-form

- Using <fieldset> + <legend> instead for <div> and <p> for checkboxes and radio.
- Use <button> than <input type="submit" /> for more flexibility.
- <div> for wrapping fields, and focus spacing in CSS instead of <br>.
